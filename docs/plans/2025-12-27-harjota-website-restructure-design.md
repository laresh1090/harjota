# Harjota Website Restructure Design

**Date:** 2025-12-27
**Status:** Validated
**Approach:** Core-First Phased Rollout
**UI Strategy:** Repurpose existing Laravel template, not rebuild

---

## Executive Summary

This document outlines the complete restructure plan for the Harjota website. The goal is to transform the current template-based site into a lead-generating, executive-focused platform that accurately represents Harjota's Institutional Intelligence services.

**Key Objectives:**
- Primary goal: Lead generation (consultation bookings)
- Target audience: C-Level executives and decision-makers
- Positioning: Industry-agnostic institutional intelligence partner
- Visual style: Premium corporate (clean, minimal, sophisticated)

---

## Strategic Decisions

| Area | Decision |
|------|----------|
| Primary Goal | Lead Generation |
| Target Audience | C-Level Executives & Decision-Makers |
| Industry Focus | Industry-Agnostic (flagship Academic Intelligence System as demo) |
| Site Structure | Streamlined (6 main pages) |
| Services Display | Tiered Packages with Pricing Visible |
| Visual Style | Premium Corporate |
| Primary CTA | Schedule a Consultation |
| Blog | Minimal initially (3-5 cornerstone articles) |
| UI Approach | Repurpose existing template, not rebuild |
| Consolidation | Merge all service/consultation pages into one Services page |
| Portfolio | Methodology Showcase (no traditional case studies) |
| SEO | Service-Specific + Local keywords |
| Contact | Multiple options (form + Calendly + WhatsApp + Email) |
| Digital Transformation | Supporting capability, not headline service |
| Social Proof | Trust indicators (methodology, credentials, process badges) |
| Timeline | Phased rollout |

## Technical Implementation

### Routes (web.php)

```php
// New Routes
Route::get('/', fn() => view('index'))->name('home');
Route::get('/about', fn() => view('about'))->name('about');
Route::get('/services', fn() => view('services'))->name('services');
Route::get('/products', fn() => view('products'))->name('products');
Route::get('/insights', fn() => view('insights'))->name('insights');
Route::get('/insights/{slug}', fn($slug) => view('insights.show', ['slug' => $slug]))->name('insights.show');
Route::get('/contact', [ContactController::class, 'showContactForm'])->name('contact');
Route::post('/contact', [ContactController::class, 'store']);

// 301 Redirects
Route::permanentRedirect('/services/business-intelligence', '/services');
Route::permanentRedirect('/services/industrial-strength', '/services');
Route::permanentRedirect('/services/cybersecurity', '/services');
Route::permanentRedirect('/services/smart-home', '/services');
Route::permanentRedirect('/services/business-development', '/services');
Route::permanentRedirect('/consultation/it-strategy', '/services');
Route::permanentRedirect('/consultation/digital-branding', '/services');
Route::permanentRedirect('/consultation/business-optimization', '/services');
Route::permanentRedirect('/consultation/cybersecurity', '/services');
Route::permanentRedirect('/portfolio', '/about');
Route::permanentRedirect('/blog', '/insights');
```
